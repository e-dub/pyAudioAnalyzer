## Installation

### Prerequisites
```
pip install scipy
pip install numpy
pip install matplotlib
pip install seaborn
pip install playsound
pip install pyfftw
pip install librosa
pip install gtts
pip install sounddevice
```

### Install
```
python setup.py install
```
